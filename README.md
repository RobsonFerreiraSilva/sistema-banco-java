# 💰 Sistema Banco em Java

Projeto desenvolvido em Java com o objetivo de simular um sistema bancário simples no terminal.

## 🚀 Funcionalidades

- ✔ Criar conta bancária  
- ✔ Depositar dinheiro  
- ✔ Sacar dinheiro  
- ✔ Consultar saldo  
- ✔ Listar todas as contas  

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Uso de classes e métodos
- ArrayList (lista dinâmica)
- Estruturas de repetição (`while`)
- Estruturas de decisão (`switch`, `if`)
- Busca de objetos na lista

## 🛠️ Tecnologias

- Java  
- IntelliJ IDEA  
- Git e GitHub  

## 📂 Estrutura do projeto

- `Main.java` → Menu e interação com o usuário  
- `Banco.java` → Gerenciamento das contas (lista e busca)  
- `ContaBancaria.java` → Representação da conta e operações
